# Welcome to Vinted

At Vinted, we strive to build solutions for business-side problems that are scalable, durable, and easy to maintain. We value code that is clean, well-tested, and flexible, and we look for colleagues who share the same mindset and engineering principles.

For this interview, you’ll have the opportunity to work with a small sample app and an existing codebase rather than starting from scratch. The interview is designed to give you context about the project and allow us to discuss code together in a collaborative way.

We’re thrilled to have you here and look forward to our conversation!

## What we’ll cover during the Interview

**Code Walkthrough**

We’ll start by showing you the app and walking you through the provided codebase so you can get familiar with the project and feature.

**Task 1: Code Review**

You’ll be presented with a Pull Request and asked to review it as you would during a standard code review.
Please walk through the code and share any comments or suggestions you find relevant.

**Task 2: Technical Discussion**

After the code review, we’ll continue with a technical discussion based on the code and related engineering topics.

**Candidate Q&A**

At the end, you’ll have time to ask any questions you may have about the role, team, company, or product.
